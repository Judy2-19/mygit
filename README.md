# CircleBallMergeGame

一个基于物理引擎的合成大西瓜类微信小游戏，玩家通过放置球体并让相同等级的球体碰撞合并，最终合成终极球获得胜利。

## 🎮 游戏玩法

1. **移动位置**：手指在屏幕上左右滑动，调整即将释放的球体位置
2. **释放球体**：松开手指，球体从顶部下落
3. **碰撞合并**：两个相同等级的球体碰撞且速度足够慢时，自动合并成更高等级的球体
4. **游戏结束**：球体越过顶部红线警戒线时游戏结束

## ✨ 功能特性

- **完整物理引擎**：实现重力、碰撞检测、分离算法、速度修正等物理模拟
- **7个等级球体**：从可爱小猫到神秘终极球，每个等级有独特颜色和动物图案
- **合并动画**：相同等级球体碰撞时触发合并动画效果
- **音效反馈**：球释放和合并时播放合成音效
- **得分系统**：实时记录并显示玩家得分
- **终极球恭喜界面**：成功合成最高等级球体时显示专属恭喜界面

## 🚀 快速开始

### 微信开发者工具

1. 打开微信开发者工具
2. 选择「导入项目」
3. 选择目录：`miniprogram/`
4. 输入AppID（测试可使用测试号）
5. 点击「导入」

### 运行游戏

- **模拟器**：直接点击「编译」按钮运行
- **真机调试**：点击「真机调试」扫描二维码在手机上运行

## 🎯 游戏规则

- 相同等级的球体碰撞且速度低于阈值时自动合并
- 合并后生成更高等级球体并获得对应得分
- 球体越过顶部红线警戒线时游戏结束
- 合成终极球（等级6）时显示恭喜界面
